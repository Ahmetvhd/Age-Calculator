# Age Calculator

Bu proje, kullanıcının doğum tarihine göre güncel yaşını yıl, ay ve gün cinsinden hesaplayan basit bir web uygulamasıdır.

## Bağlantılar

- **Canlı Önizleme:** [https://ahmetvhd.github.io/Age-Calculator/](https://ahmetvhd.github.io/Age-Calculator/)
- **Proje Kaynağı:** [roadmap.sh - Age Calculator](https://roadmap.sh/projects/age-calculator)

## Özellikler

- Doğum tarihi girişi ile yaş hesaplama.
- Eksik veya hatalı girişler için doğrulama kontrolleri.
- Responsive (mobil uyumlu) tasarım.
